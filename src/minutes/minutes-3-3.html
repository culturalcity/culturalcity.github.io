參、議題討論：

議案議案
114年度物業換約案討論
說明

目前社區之物業服務合約期間至113年12/31止，新美齊物業管理維護費/月 $286,650。

決議

因近年勞動市場人力成本都有一定比例調幅，因此擬定於113年11月底前進行物業遴選，研討決議新物業合約。

議案議案
114年度園藝保養換約案討論
說明

園藝養護合約期間至113年11/30止，養護費/月 $7,500

決議

經觀察原合約廠商(綠石園藝)之養護施工品質合格，決議續約，請綠石提供114年合約並商議希望維持原合約報價。

議案議案
113年公設冷氣保養清洗討論
說明

因避免年後討論決議施工，恐面臨廠商施工排程延遲過久。目前請三家廠商報價如下 :

1). 晉翊空調 $87,700 (未稅)

2). 翔立空調 $66,500 (未稅)

3). 益誠空調 $59,700 (未稅，去年維護廠商)

決議

由(益誠空調)承接公設空調保養工程，費用$59,700 (未稅)

肆、臨時動議：

動議一、

1F之6永慶裝修驗收狀況: 外陽台冷氣壓縮機裸露過高

說 明 :

討論1). 請永慶改善冷氣壓縮機裸露過高問題回覆，是否參

照隔壁牙醫店面一樣高度即可?

討論2). 因冷氣安裝工程為永慶另尋廠商建置；非原申請裝

修之裝潢公司執行，目前已完成裝修驗退檢查，其保

證金是否先受理退款給予申請裝修的廠商。

決 議 :

討論1).同意，壓縮機降低至與隔壁牙醫店面一樣高度即可。

討論2).待壓縮機降低高度驗收後才能受理驗退保證金。

動議二、

清潔員休假日應補代班清潔員執勤

說 明 :

住戶反應，每月清潔員休假日8~9天(不含連續假期)，是否增加代班清潔員，避免多日無人打掃與維護。

決 議 :

暫不增加清潔員休假日增加代班人力，先落實依排程實際完成工作項目，後續觀察需求量再討論是否增加人力。

動議三、

RF戶外石桌桌板更新材質討論

說 明 :

防颱期間搬運石桌板時破裂，建設公司建議兩種材質 :

1). 只換岩板桌 : 檯 面$18,000未稅

2). 鋁製 / 整組 : 桌與板$29,800未稅

決 議 :

1). 只換岩板桌 : 檯 面$18,000未稅。

動議四、

是否開放1F永慶員工租社區機車停車位

說 明 :

1F永慶員工詢問，需要租社區停車場機車位。

動議五、

風災後：

15樓、4樓梯廳漏水問題。

說 明 :

各樓層的排煙窗均有明顯滲水，風力強時高樓層受鋒面大，如周圍無阻擋時更顯著，滲水量已危及流入電梯。

決 議 :

建設公司將派員至現場勘查修復。
