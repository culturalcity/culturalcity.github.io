# 為什麼要做這個

閱大安是**列冊的公共防空避難處所**（門牌處設有內政部「防空避難標誌牌」）。2026/08/13 台北市舉行「城鎮韌性（全民防空）演習」，依《民防法》第 21 條、《防空演習實施辦法》第 7、12 條，大廈管理人於演習時**有配合開放地下室避難設施、供民眾就近避難的義務**；未配合者，得依《民防法》第 25 條處 **3 萬～15 萬元** 罰鍰。

挑戰是：**既要配合開放供公眾避難，又要守住社區門禁安全。** 做法是「分層＋戒護＋復歸」，兩者可以兼顧，不必為演習犧牲住宅安全。

> 這件事**不對住戶公告**（住戶端只宣導「聽到警報怎麼避難」）；開放屬管委會內部執行，故收在此頁供總幹事／保全依循。

# 演習關鍵時間（2026/08/13 週四）

| 項目 | 內容 |
|---|---|
| 管制時段 | **14:30–15:00**（警報傳遞＋疏散避難＋交通管制，共 30 分鐘）|
| 空襲警報 | 1 長音 ＋ 2 短音，連續 3 次 |
| 解除警報 | 1 長音，鳴放 90 秒 |
| 方式 | 有預警（事前公告的例行演習）|

# 門禁戒護原則：把大樓想成「兩層殼」

1. **外殼＝地下室停車場**（演習要開放的避難區）→ 依訓令開啟車道鐵捲門
2. **內殼＝電梯／住宅樓層門禁** → **全程維持上鎖**

開放的是外殼；進來避難的人到停車場層蹲低避難即可，**上不了樓、進不了住家**。動到的是停車場，不是住戶的家。

# 當天流程

| 時間 | 動作 |
|---|---|
| **前一日 8/12** | 確認：① 地下室避難區淨空、動線與標示清楚 ② 車道鐵捲門手動／自動切換正常、逃生小門正常 ③ 手電筒／哨子等就位 |
| **14:15 就位** | 保全至地下室車道開口待命；再次確認**電梯與住宅樓層門禁維持上鎖** |
| **14:30 開放** | 依現場民防人員訓令**開啟車道鐵捲門**；引導避難民眾至停車場**指定避難區**，明確**阻擋往電梯／住宅動線** |
| **演習中** | 維持秩序、留意進入人數；勿讓任何人上樓；聽從民防人員指示 |
| **15:00 解除警報後** | 引導民眾離場 → 清點 → **鐵捲門復歸** → 確認門禁全數復原 |
| **事後** | 回報主委：當天是否有民眾進入、進入人數概估、有無任何異常 |

# 注意事項

- **鐵捲門復歸要小心**：參考先前 [地下停車場防火鐵捲門「無火警誤降」事件](/admin/staff/docs/2026-05-06-fire-shutter-incident/) 的教訓——降門前務必確認**逃生小門先關、門面完整**再落下，避免門片脫軌外凸。
- 若演習前收到警察局／民防管制中心的**個別通知或更細的訓令**，一律以其為準。
- 民防相關疑義：台北市警察局民防管制中心 **(02) 8861-1124**。
